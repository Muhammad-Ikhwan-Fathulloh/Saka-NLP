Saka NLP

Pilar pemrosesan bahasa Nusantara yang modern. Architectural framework solid untuk Bahasa Indonesia dan daerah dengan dukungan asynchronous native.

📦 PyPI Package 📓 Google Colab ⭐ GitHub

Pusat Dokumentasi

Cek Versi

Pastikan Anda menggunakan versi terbaru untuk mendapatkan fitur pilar terbaru.

Python 
import saka

print(saka.__version__)
# Output: 0.1.4

Tokenisasi Cerdas

Memecah kalimat menjadi satuan kata secara presisi.

Python 
import saka

text = "Belajar sambil beramal di era konektivitas."
tokens = saka.tokenize(text)
print(tokens)
# Output: ['Belajar', 'sambil', 'beramal', 'di', 'era', 'konektivitas']

Normalisasi Slang

Mengubah bahasa gaul sosial media menjadi bahasa formal secara otomatis.

Python 
import saka

normalized = saka.normalize("klo gimana gw")
print(normalized)
# Output: 'kalau bagaimana saya'

Analisis Morfologi

Bedah struktur kata hingga ke akar (*kata dasar*) beserta prefiks dan sufiksnya.

Python 
import saka

analysis = saka.analyze("mempertanggungjawabkan")
print(analysis)
# Output:
# {
#   'root': 'tanggung jawab',
#   'prefixes': ['mem', 'per'],
#   'suffixes': ['kan'],
#   'type': 'unknown'
# }

Live KBBI Search

Integrasi langsung dengan database KBBI Online untuk pencarian definisi kata.

Python 
import saka

kbbi_result = saka.query_kbbi("belajar")

if kbbi_result["status"] == "found":
    for arti in kbbi_result["definitions"]:
        print(f"Arti: {arti}")

Native Stopwords

Akses cepat ke ribuan kata umum Indonesia untuk kebutuhan filter teks.

Python 
import saka

stopwords = saka.get_stopwords()
print(f"Apakah 'ada' adalah stopword? {'ada' in stopwords}")

Bahasa Daerah: Sunda

Dukungan kamus SundaDigi dan transliterasi Aksara Sunda yang presisi secara fonetik.

Python 
import saka

# Kamus SundaDigi
res = saka.query_sundadigi("wilujeng")

# Aksara Sunda
aksara = saka.latin_to_aksara_sunda("saka")
print(aksara) # Output: ᮞᮊ
